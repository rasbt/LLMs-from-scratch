# Claude Code Docs (/docs)

Source: https://code.claude.com/docs

## TL;DR
- Terminal-first coding agent by Anthropic.
- Install via curl/PowerShell/WinGet; run `claude` inside a repo.
- Key strengths: action-taking, composability, MCP integrations.

## Quickstart
Prerequisites: Claude subscription or Claude Console account.

Install:
- macOS/Linux/WSL: curl -fsSL https://claude.ai/install.sh | bash
- Windows PowerShell: irm https://claude.ai/install.ps1 | iex
- Windows CMD: curl -fsSL https://claude.ai/install.cmd -o install.cmd && install.cmd && del install.cmd

Start:
- cd your-project
- claude

Notes: Native installs auto-update. See setup/troubleshooting for advanced options.
